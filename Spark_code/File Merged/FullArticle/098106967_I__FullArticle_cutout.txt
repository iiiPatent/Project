 發明 領域 
 本 發明 係 關於 保護 水果 、 蔬菜 及 觀賞 用 植物 免於 受致 腐壞 有機體 諸如 真菌 攻擊 。 特定 言 之 ， 本 發明 係 關於 經由 施用 系統性 抵抗力 誘導劑 組合 山梨酸 及 / 或 其 鹼金屬鹽 之 混合物 而 保護 水果 或 蔬菜 或 觀賞 用 植物 免於 收穫 後 微生物 性 腐壞 之 方法 。 
 發明 背景 
 於 水果 、 蔬菜 及 觀賞 用 植物 加工處理 中 ， 施用 抗微生物劑 至新鮮 收穫 的 水果 、 蔬菜 及 觀賞 用 植物 表面 用於控 制致 腐壞 有機體 為 眾所周知 的 食物 。 
 系統性 抵抗力 誘導劑 已知 可 用於植物 提 引出 微生物 抵抗力 ， 但 未曾 用於 水果 、 蔬菜 及 觀賞 用 植物 之 收穫 後 用途 ， 原因 在於 其 未能 以 可接受的 程度 提供 微生物的 控制 緣故 。 
 已知 於 水果 、 蔬菜 及 觀賞 用 植物 中 山梨酸鹽 可 提供 對抗 微生物分解 之 若干 保護 程度 ， 但當 單獨 施用 時無法 以 可接受的 商業 程度 提供 保護 。 
 隨著 收穫 後 水果與蔬菜 區塊 方面 法規 與 環保 壓力 的 增高 ， 多種 抗微生物 材料 由 市場 撤出 或 無法 再度 註冊 登記 。 因此 期望 提供 使用方便 易得 、 毒性 低且 不 可能 被 禁止 註冊 登記 之 材料 來 保護 水果 或 蔬菜 或 觀賞 用 植物 免於 收穫 後 微生物 腐壞 之 方法 。 
 本 發明 提供 利用 具有 極低 哺乳動物 毒性 且 對 環保 友善 之 材料 來 減少 水果 、 蔬菜 及 觀賞 用 植物 之 微生物 性 腐壞 之 新穎方法 。 
 發明 概要 
 本 發明 包括 經由 施用 系統性 後天性抵抗力 誘導劑 組合 山梨酸 及 / 或 其 鹼金屬鹽 來 保護 水果 、 蔬菜 及 / 或 觀賞 用 植物 免於 收穫 後 微生物 性 腐壞 之 方法 。 於 若干 實施例 中 ， 施用 至 水果 及 / 或 蔬菜 之 材料 包括 山梨酸 及 / 或 其 鹼金屬鹽 以及 亞磷酸 及 / 或 其 鹼金屬鹽 及 / 或 銨鹽 。 已經 測定 單獨 山梨酸 及 / 或 其鹽 有助於 保護 水果 及 蔬菜 免於 收穫 後 腐壞 。 也 已經 測定 單獨 亞磷酸 及 / 或 其鹽 也 可 有助於 保護 水果 及 蔬菜 免於 收穫 後 腐壞 。 
 山梨酸 ( 及 / 或 其鹽 ) 與 亞磷酸 ( 及 / 或 其鹽 ) 組合 用於 減少 收穫 後 腐壞 之 效果 發現 係 大於 單獨 山梨酸 ( 及 / 或 其鹽 ) 或 單獨 亞磷酸 ( 及 / 或 其鹽 ) 用於 低溫 或 高溫 減少 收穫 後 腐壞 之 個別效果 之 總和 。
