渲染 皂 成形 方法 、 使用 工具 、 及其 所 製得 之 渲染 皂 
 本 發明 目的在於 提供 一種 色彩 多樣 、 製造 便利 、 渲染效果 多元化 的 渲染 皂 成形 方法 、 使用 工具 、 及其 所 製得 之 渲染 皂 。 其 技術 手段 為 ： 一皂模 ( 1 ) ， 該皂模 ( 1 ) 是 由 一內 設有 第一 容 置空間 ( 13 ) 、 頂 端設有 一 開口 ( 11 ) 的 外皂模 ( 12 ) ， 及 一 套設 於 該 第一 容 置空間 ( 13 ) 內 、 並內 設有 第二 容 置空間 ( 15 ) 、 且 表面 光滑的 內皂模 ( 14 ) 所 組成 ； 以及 一設 於 該 第二 容 置空間 ( 15 ) 內的 分隔 裝置 ( 2 ) ， 該 分隔 裝置 ( 2 ) 是 由 至少 一個以上 相對 設 於 該 第二 容 置空間 ( 15 ) 內的 隔板 骨架 ( 21 ) ， 及 至少 一個以上 設 於 該 隔板 骨架 ( 21 ) 間 、 將 相對 隔板 骨架 ( 21 ) 連接 、 並能將 第二 容 置空間 ( 15 ) 分為 數個 溝槽 空間 ( 400 ) 的 隔板 ( 22 ) 所 組成 。 
 本 發明 是 涉及 一種 渲染 皂 成形 方法 、 使用 工具 、 及其 所 製得 之 渲染 皂 。 
 在 18世紀 末 工業革命 工業 問世 後 ， 人類 獲得 了 大量的 價廉 的 碳酸鈉 ， 促使 肥皂 工業 有 了 全新的 發展 。 
 但是 ， 到了 20世紀 中期 ， 合成化學 和 石油化工 的 發展 ， 為 洗滌劑 提供 了 廉價 的 化工原料 ， 促使 了 合成洗滌劑 的 興起 ， 使得 肥皂 工業 的 發展 ， 發生 了 很大 的 變化 。 
 但是 ， 近年 來 環保意識 的 抬頭 ， 己 被 大眾 所 忽略 的 手工皂 ， 才 又 開始 興起 ， 手工肥皂 則 運用 了 油脂的 皂化反應 ， 所以 能 保留 天然 保濕劑 - 甘油 ( Glycerin ) 的 滋潤 ， 且 與 工廠 大量生產的 香皂 不同 ， 其 成份 內 沒有 添加 界面活性劑 、 起泡劑 、 防腐劑 、 無淚劑 等 化學物質 ， 對 人體無 害 。 
 手工皂 的 優點 很多 ， 除了 其中 含有 1 / 4 以上 ， 對 肌膚 洗淨 溫 和 、 且 保濕 的 甘油 ， 而且 其 手工皂 的 材料 「 油 、 鹼 、 添加物 」 ， 均 為 天然 素材 ， 自然的 素材 ， 其 整體的 美容 保養 效果 ， 並不會 比 化學品 差 ， 而且 更 不易 引起 過敏 ， 所以 很 適合 敏感 肌膚 的人 使用 。 
 另外 ， 因為 手工皂 跟 化學 清潔劑 、 市售 香皂 不同 ( 不 含 化學成份 ) ， 手工皂 有 其 天然 特有的 性能 ， 在 與水 接觸 後 大約 24小時 ， 其 內的 各 種組成 與 成份 ， 很 容易 就 會 被 生物降解 ， 並易 於 被 污水處理過程 中的 微生物分解 ， 因此 ， 不會 引起 河流 、 湖泊 和 水道 的 污染問題 ， 不 影響 水 世界的 生態 ， 所以 多用 手工皂 ， 還 可以 為 環保 盡 一份 心力 。 
 此外 ， 一般 手工皂 分為 四種 ， 如 下列 所述 ： 熱 製法 ( Hot   Process ) ， 屬於 持續性 的 加熱 ， 它 的 好處 是 製 皂 時間 很 短 ， 約二個 小時 即可 完成 ， 缺點 是 部分 營養 都 在 持續性 的 高 溫下 被 破壞 了 。 
 冷製法 ( Cold   Process ) ， 可以 保留 高級 植物油 的 維他命 營養 、 及 精油 的 成分 ， 最重要的 莫過 於 含有 甘油 ， 能滋潤 保濕 使用者 的 肌膚 ， 但 缺點 是 製作 時 間極 長 ， 最少 必須 放置 於 自然環境中 ， 等待 4 ～ 8 週 以上 才 可以 皂化 完成 。 
 再生法 ( Re batching ) ， 冷 製 皂 刨 絲 加熱 融化 、 重新 塑模 ， 此種 方法 做成 的 皂 為 「 再生 皂 」 ， 完成 後 需 放置 晾乾 ， 有時 需達 3 星期 以上 。 另外 市面上 也 可 買 到 現成的 橄欖油皂 絲 、 椰子油 皂絲 ， 可以 加熱 溶化 塑型 。 
 融化 再 製法 ( Melt   &   Po ur ) ， 是 將市售 現成 皂基 ， 隔 水加熱 溶化 後 ， 添加 色素 和 香味 ， 重新 塑型 的 皂 ， 脫膜 後 馬上 即可 使用 。 
 而 其中 ， 冷製法 的 手工皂 中 ， 更 包含 了 一種 渲染 皂 ， 透過 單色 或 雙色的 皂液 ， 進行 渲染 ， 凝固 後 脫模 ， 經過切割 後 ， 能 於 斷面 處 ， 呈現出 大理石花紋 般的 藝術效果 ， 且 因為 是 手工製 作 ， 圖形 難以 控制 ， 也 因此 每 一次 都 能 有 不同的 花紋 、 不同的 感受 ， 是 目前 最受 青睞 的 手工皂 。 
 前述 習知 渲染 皂 製法 、 工具 與 渲染 皂 存在 著 下列 問題點 ： 
 1 . 習知 渲染 皂 製法 ， 其 最大的 問題點 就是 ， 受到 冷製法 的 影響 ， 能 製作 的 渲染 皂 的 顏色變化 有限 ， 最多為 雙色 ， 也 就是 加上 皂 體的 顏色 後 ， 最 多 只有 三種 色 ， 讓 使用者 只能 專注 於 花紋 的 變化 ， 產生的 產品 ， 相較 於 複雜 、 困難的 製法 ， 對 初學者 的 吸引力 有限 。 
 有鑑 於 此 ， 如何 讓 渲染 皂 製法 能 適用於 ， 製作 多色的 渲染 皂 ， 且 降 低複雜度 與 難度 ， 便 成為 本 發明 欲 改進的 目的 之一 。 
 2 . 習知 渲染 皂 工具 ， 其 最大的 問題點 就是 ， 一般 只能 做到 雙色 渲染 皂 的 應用 ， 使用範圍 有限 ， 且 在 製作 過程中 ， 容易 發生 不穩定的 現象 ， 而 讓 渲染 皂 失敗 ， 另一方面 ， 在 使用 上 也 較 不 方便 ， 且 整體的 而言 ， 要 做出 好看 的 渲染 皂 ， 程度 上 亦 較 其他的 手工皂 困難 ， 需要 有 一定的 藝術 天份 。 
 有鑑 於 此 ， 如何 讓 應用範圍 增加 、 成功率 變高 、 使用 更 方便 ， 並讓 所有人 都 能 容易 上 手 ， 做出 好看 的 渲染 皂 ， 便 成為 本 發明 欲 改進的 目的 之二 。 
 3. 習知 渲染 皂 ， 其 最大的 問題點 就是 ， 受到 製法 與 工具的 影響 ， 能 產生的 顏色 效果 有限 ， 分別 為 單色 或 雙色 ， 與 其他 種類 的 手工皂 相 比較 之下 ， 只有 其 花紋 較 為 特別 ， 而 其他 方面 則較 為 遜色 ， 且 整體的 美觀 度 有限 ， 在 效果 相同的 狀態 下 ， 會 影響 其 產業利用性 。 
 有鑑 於 此 ， 如何 增加 顏色 效果 ， 並 配合 花紋 ， 提高 整體 美觀 度 ， 增加 其 產業利用性 ， 便 成為 本 發明 欲 改進的 目的 之三 。 
 
 本 發明 目的在於 提供 一種 色彩 多樣 、 製造 便利 、 渲染效果 多元化 的 渲染 皂 成形 方法 、 使用 工具 、 及其 所 製得 之 渲染 皂 。 
 為 解決 前述 問題 及 達到 本 發明 的 目的 ， 本 發明 技術 手段 ， 在 成形工具 方面 ， 其 技術 手段 有 以下 兩種 實施 型態 ： 本 發明 第一 型態 ： 為 一種 渲染 皂 成形工具 ( 100 ) ， 其特 徵在於 包括 ： 一皂模 ( 1 ) ， 該皂模 ( 1 ) 是 由 一內 設有 第一 容 置空間 ( 13 ) 、 頂 端設有 一 開口 ( 11 ) 的 外皂模 ( 12 ) ， 及 一 套設 於 該 第一 容 置空間 ( 13 ) 內 、 並內 設有 第二 容 置空間 ( 15 ) 、 且 表面 光滑的 內皂模 ( 14 ) 所 組成 ； 以及 一設 於 該 第二 容 置空間 ( 15 ) 內的 分隔 裝置 ( 2 ) ， 該 分隔 裝置 ( 2 ) 是 由 至少 一個以上 相對 設 於 該 第二 容 置空間 ( 15 ) 內的 隔板 骨架 ( 21 ) ， 及 至少 一個以上 設 於 該 隔板 骨架 ( 21 ) 間 、 將 相對 隔板 骨架 ( 21 ) 連接 、 並能將 第二 容 置空間 ( 15 ) 分為 數個 溝槽 空間 ( 400 ) 的 隔板 ( 22 ) 所 組成 。 
 根據 上述 的 渲染 皂 成形工具 ( 100 ) ， 所述 外皂模 ( 12 ) 的 材質 ， 是 為 下列 之一 ： 壓克力 材 、 木材 、 硬質 塑膠 材 、 金屬材 ； 而 所 述內 皂模 ( 14 ) 是 為 矽膠 材質 。 
 根據 上述 的 渲染 皂 成形工具 ( 100 ) ， 所述 隔板 骨架 ( 21 ) 的 頂端 ， 相對 設有 數個 間隔 設置 、 能供 隔板 ( 22 ) 卡接 的 凹槽 ( 23 ) ； 而 該 隔板 ( 22 ) 與 凹槽 ( 23 ) 連接處 ， 各 設有 一能 與 凹槽 ( 23 ) 配合 、 並 凸出 於 皂模 ( 1 ) 的 凸 楯 ( 24 ) 。 
 根據 上述 的 渲染 皂 成形工具 ( 100 ) ， 所述 隔板 骨架 ( 21 ) 及 隔板 ( 22 ) 的 材質 ， 是 為 下列 之一 ： 壓克力 材 、 硬質 塑膠 材 。 
 根據 上述 的 渲染 皂 成形工具 ( 100 ) ， 所述 皂模 ( 1 ) 的 開口 ( 11 ) ， 更 設有 一能將 第二 容 置空間 ( 15 ) 封閉的 頂蓋 ( 16 ) 。 
 本 發明 第二型態 ： 為 一種 渲染 皂 成形工具 ( 100 ) ， 其特 徵在於 包括 ： 一 頂端 為 開口 ( 11 ) 的 皂模 ( 1 ) ， 該皂模 ( 1 ) 是 由 一 底座 ( 17 ) 、 及 一端 設 於 該 底座 ( 17 ) 內 、 呈 中空狀 、 可拆卸 式 的 一立壁 殼體 ( 18 ) 所 組成 ， 該 底座 ( 17 ) 與 立壁 殼體 ( 18 ) ， 圍繞 形成 一 第三 容納 空間 ( 19 ) ， 而 該 立壁 殼體 ( 18 ) 的 兩頂 相 對端 ， 各 設有 一 呈 齒狀的 卡接部 ( 18 1 ) ； 以及 一設 於 該 第三 容 置空間 ( 19 ) 內的 分隔 裝置 ( 2 ) ， 該 分隔 裝置 ( 2 ) 是 由 至少 一個以上 設 於 該 第三 容 置空間 ( 19 ) 內 、 兩端 分別 連接 卡接部 ( 18 1 ) 、 並能將 第三 容 置空間 ( 19 ) 分為 數個 溝槽 空間 ( 400 ) 的 隔板 ( 22 ) 所 組成 。 
 根據 上述 的 渲染 皂 成形工具 ( 100 ) ， 所述 所述 皂模 ( 1 ) 的 材質 ， 是 為 下列 之一 ： 壓克力 材 、 硬質 塑膠 材 ； 而 隔板 ( 22 ) 的 材質 ， 是 為 下列 之一 ： 壓克力 材 、 硬質 塑膠 材 。 
 根據 上述 的 渲染 皂 成形工具 ( 100 ) ， 所述 該 隔板 ( 22 ) 與 卡接部 ( 18 1 ) 的 連接處 ， 設有 能 與 卡接部 ( 18 1 ) 配合 、 並 凸出 於 皂模 ( 1 ) 的 凸 楯 ( 24 ) 。 
 根據 上述 的 渲染 皂 成形工具 ( 100 ) ， 所述 皂模 ( 1 ) 的 開口 ( 11 ) 處 ， 更 設有 一能將 第三 容 置空間 ( 19 ) 封閉的 頂蓋 ( 16 ) 。 
 解決 前述 問題 及 達到 本 發明 的 目的 ， 本 發明 技術 手段 在 製造方法 方面 ， 為 一種 渲染 皂 成形 方法 ， 其特 徵在於 包括 ： 
 步驟 一 ： 將 調配 好的 初始 皂液 ( 37 ) ， 以 攪拌器 ( 4 ) ， 打至 呈現 濃稠狀 ， 使 其 進行 皂化反應 ， 形成 皂液 ( 3 ) ( A ) ； 
 步驟 二 ： 皂液 ( 3 ) 分 為數 份 ， 並 分別 加入 色素 ( 6 ) ， 調成 不同顏色 ， 並 攪拌 均勻 ( B ) ； 
 步驟 三 ： 將已 調色 的 皂液 ( 3 ) ， 分別 倒入 渲染 皂 成形工具 ( 100 ) 的 溝槽 空間 ( 400 ) 中 ( C ) ； 
 步驟 四 ： 皂液 ( 3 ) 完全 倒入 溝槽 空間 ( 400 ) 後 ， 再 依序 抽出 隔板 ( 22 ) ( D ) ； 
 步驟 五 ： 完全 抽出 隔板 ( 22 ) 後 ， 再利用 一 渲染 棒 ( 5 ) ， 進行 攪拌 ， 開始 劃出 渲染 圖 ( E ) ； 
 步驟 六 ： 將 頂蓋 ( 16 ) 蓋 闔 上 ， 以 讓 皂液 ( 3 ) 保溫 ， 並 靜置 一段 預定時間 ， 讓 皂液 ( 3 ) 凝固 成形 為 一 初始 渲染 皂 ( 200 ) ( F ) ； 
 步驟 七 ： 將已 凝固的 初始 渲染 皂 ( 200 ) ， 自 渲染 皂 成形工具 ( 100 ) 中 取出 ( G ) ； 以及 
 步驟 八 ： 將 取出 的 初始 渲染 皂 ( 200 ) ， 進行 切皂 的 動作 ， 成為 數個 渲染 皂 ( 30 0 ) ( H ) 。 
 根據 上述 的 渲染 皂 成形 方法 ， 所述 初始 皂液 ( 37 ) ， 是 由 一以 內含 氫氧化鈉 ( 71 ) 和 純水 ( 72 ) 、 溫度 為 40 ～ 60 ℃ 的 鹼水 ( 7 ) ； 以及 一溫度 為 40 ～ 60 ℃ 、 完全 融化 的 油脂 ( 8 ) 所 混合 而成 。 
 根據 上述 的 渲染 皂 成形 方法 ， 所述 渲染 皂 成形工具 ( 100 ) ， 是 為如 申請專利範圍 第1 項 所述 的 渲染 皂 成形工具 ( 100 ) ； 而 該 步驟 五中 ， 當 渲染 皂 成形工具 ( 100 ) 是 為如 申請專利範圍 第1 項 所述 的 渲染 皂 成形工具 ( 100 ) 時 ， 完全 抽出 隔板 ( 22 ) 後 ， 更 包括 一將 隔板 骨架 ( 21 ) 完全 抽出 的 步驟 。 
 根據 上述 的 渲染 皂 成形 方法 ， 所述 渲染 皂 成形工具 ( 100 ) ， 是 為如 申請專利範圍 第 6 項 所述 的 渲染 皂 成形工具 ( 100 ) 。 
 根據 上述 的 渲染 皂 成形 方法 ， 所述 步驟 八中 的 切皂 動作 ， 是 為 先將 初始 渲染 皂 ( 200 ) 粗 分段 ， 再行 橫剖 ， 其橫剖 方向 與 倒入 皂液的 方向 ， 是 為 垂直 。 
 根據 上述 的 渲染 皂 成形 方法 ， 所述 色素 ( 6 ) 是 為 下列 之一 ： 礦泥 粉 、 皂用 色素 、 植物 顏料 、 食用色素 、 中藥草 粉 、 咖啡粉 。 
 根據 上述 的 渲染 皂 成形 方法 ， 所述 攪拌器 ( 4 ) 是 為 下列 之一 ： 電動 打蛋器 、 手動 打蛋器 。 
 根據 上述 的 渲染 皂 成形 方法 ， 所述 渲染 棒 ( 5 ) 是 為 下列 之一 ： 塑膠棒 、 木頭 棒 、 竹棒 、 金屬棒 。 
 本 發明 技術 手段 ， 在 製成的 成品 方面 ， 為 一種 渲染 皂 ， 其特 徵在於 包括 ： 一以 申請專利範圍 第 10 項 所述 之 方法 所 製成 、 且 切皂後能 於 斷面 處 、 呈 對稱 狀 花紋 、 為 多彩的 渲染 皂 ( 30 0 ) 。 三 、 對照 先前技術 之 功效 ： 
 1 . 本 發明 中 ， 最大的 特點 在於 ， 在 製造過程 中 ， 是 使用 本 發明 所述 的 渲染 皂 成形工具 ( 100 ) ， 做 為 渲染 皂 ( 30 0 ) 的 成形工具 ， 其 主要 原理 ， 透過 多個 隔板 ( 22 ) ， 來達 到將 多種顏色 的 皂液 ( 3 ) ， 分出 層次的 效果 ， 當 皂液 ( 3 ) 全數 倒入 後 ， 只要 將 隔板 ( 22 ) 及 隔板 骨架 ( 21 ) 都 取出 後 ， 即能 看到 此一 分層 效果 ， 另外 ， 因為 皂液 ( 3 ) 的 黏稠度 夠 ， 所以 不會 有 互溶 的 狀況 發生 ， 所以 此時 只要 透過 渲染 棒 ( 5 ) ， 將其 攪動 ， 便 能 進行 渲染 的 動作 ， 當 皂液 ( 3 ) 凝固 ， 取出 初始 渲染 皂 ( 200 ) ， 並 進行 切皂 的 動作 後 ， 所 獲得的 渲染 皂 ( 30 0 ) ， 便 能 呈現出 多彩的 對稱 狀 花紋 ， 與 傳統 渲染 皂 的 製造 方式 大不相同 ， 色彩 豐富 而 豔 麗 ， 且 操作 上 更 為 方便 、 自在 。 
 2 . 本 發明 中 ， 透過 渲染 皂 成形工具 ( 100 ) 的 使用 ， 更能 方便 使用者 利用 ， 與 傳統 渲染 皂 的 製作 工具 不同 ， 使用者 能夠 很 簡單的 做出 ， 所需要 的 渲染效果 ， 能 提高 渲染 皂 ( 30 0 ) 的 成功機率 ， 降低 渲染 的 難度 ， 能 增加 消費者 的 購買意願 ， 因此 產業利用性 佳 、 實用性 高 。 
 3. 以本 發明 中的 方法 ， 所 生產 出來 的 渲染 皂 ( 30 0 ) ， 其 顏色 效果 的 多寡 ， 是 依 隔板 ( 22 ) 的 間隔數 量 來 決定 ， 所以 能 有 非常 多元化 的 色彩 效果 ， 能夠 配合 花紋 ， 產生 更 多樣化 的 圖樣 ， 提高 整體 美觀 度 ， 增加 其 產業利用性 。 
 4. 本 發明 中 ， 將 隔板 ( 22 ) 去除 後 ， 不 進行 渲染 的 動作 ， 還可 產生 另 一種 冷 製 皂 ， 也 就是 「 分層 皂 」 ， 而且 製造 上 ， 較 傳統 分層 皂 製法 ， 操作 上 更 方便 ， 容易 上 手 ， 讓 整體的 應用範圍 更加 的 廣泛 ， 實用性 更高 。

 第一 實施 型態 ： 
 圖式 中 揭示出 ， 為 一種 渲染 皂 成形工具 ( 100 ) ， 其特 徵在於 包括 ： 一皂模 ( 1 ) ， 該皂模 ( 1 ) 是 由 一內 設有 第一 容 置空間 ( 13 ) 、 頂 端設有 一 開口 ( 11 ) 的 外皂模 ( 12 ) ， 及 一 套設 於 該 第一 容 置空間 ( 13 ) 內 、 並內 設有 第二 容 置空間 ( 15 ) 、 且 表面 光滑的 內皂模 ( 14 ) 所 組成 ； 以及 一設 於 該 第二 容 置空間 ( 15 ) 內的 分隔 裝置 ( 2 ) ， 該 分隔 裝置 ( 2 ) 是 由 至少 一個以上 相對 設 於 該 第二 容 置空間 ( 15 ) 內的 隔板 骨架 ( 21 ) ， 及 至少 一個以上 設 於 該 隔板 骨架 ( 21 ) 間 、 將 相對 隔板 骨架 ( 21 ) 連接 、 並能將 第二 容 置空間 ( 15 ) 分為 數個 溝槽 空間 ( 400 ) 的 隔板 ( 22 ) 所 組成 。 
 其中 ， 透過 具有 多個 隔板 ( 22 ) 的 分隔 裝置 ( 2 ) ， 來將 多種顏色 的 皂液 ( 3 ) 隔離 ， 讓 皂液 ( 3 ) 能 在 不 互相干擾 、 擾動 的 狀態 下 ， 分出 層次 ， 而當 皂液 ( 3 ) 全數 倒入 後 ， 等待 皂液 ( 3 ) 靜止 後 ， 便 能將 隔板 ( 22 ) 及 隔板 骨架 ( 21 ) 都 取出 ， 即能 達成 皂液 ( 3 ) 分層的 效果 ， 另一方面 ， 因為 皂液 ( 3 ) 的 黏稠度 夠 ， 所以 不會 有 互溶 的 狀況 發生 ， 而 此時 要 趁 著 ， 皂液 ( 3 ) 還未 凝固 時 ， 透過 渲染 棒 ( 5 ) ， 將其 攪動 ， 以 進行 渲染 的 動作 ， 隨後 當 皂液 ( 3 ) 凝固 ， 便 可 自皂模 ( 1 ) 中 ， 取出 一 初始 渲染 皂 ( 200 ) ， 進行 切皂 後 ， 即可 獲得 呈現出 多彩的 對稱 狀 花紋 的 渲染 皂 ( 30 0 ) ， 操作 上 較 與 傳統 渲染 皂 的 製造工具 ， 更為 方便使用 。 
 其次 ， 因為 是 藉由 隔板 ( 22 ) 的 數量 ， 來 決定 ， 所以 能 使用 色彩的 數量 ， 因此 本 發明 能 有眾 多的 變化 ， 且 因為 使用 上 更 為 方便 ， 所以 使用者 能 更 容易的 自在 創作 ， 不必 擔心 會 失敗 。 
 再者 ， 整體的 主要構造 ， 分為 皂模 ( 1 ) 與 分隔 裝置 ( 2 ) 兩 部分 ， 能 方便 收納 ， 而且 成本 不會 太高 ， 配合 使用 上的 便利性 ， 能 增加 消費者 的 購買意願 。 
 上述 中 ， 所述 外皂模 ( 12 ) 的 材質 ， 是 為 下列 之一 ： 壓克力 材 、 木材 、 硬質 塑膠 材 、 金屬材 ； 而 所 述內 皂模 ( 14 ) 是 為 矽膠 材質 。 
 其中 ， 使用 不同的 外皂模 ( 12 ) ， 能 符合 不同 使用者 的 需求 ， 並能 應對 不同尺寸 的 內皂模 ( 14 ) ， 提供 足夠的 支撐力 。 
 其次 ， 為矽 膠材 質的 內皂模 ( 14 ) ， 其 內表面 有 足夠的 光滑度 ， 且 因為 矽膠 材質 是 軟性 體 ， 並能 重覆使用 ， 結合 此 三個 因素 ， 能 有利於 脫模 作業 ， 與 後續 的 再利用 。 
 上述 中 ， 所述 隔板 骨架 ( 21 ) 的 頂端 ， 相對 設有 數個 間隔 設置 、 能供 隔板 ( 22 ) 卡接 的 凹槽 ( 23 ) ； 而 該 隔板 ( 22 ) 與 凹槽 ( 23 ) 連接處 ， 各 設有 一能 與 凹槽 ( 23 ) 配合 、 並 凸出 於 皂模 ( 1 ) 的 凸 楯 ( 24 ) 。 
 其中 ， 透過 凹槽 ( 23 ) 的 設置 ， 能 配合 凸出 於 皂模 ( 1 ) 的 凸 楯 ( 24 ) ， 讓 隔板 ( 22 ) 能 不 被 倒入 的 皂液 ( 3 ) 影響 ， 能 穩固的 將 第二 容 置空間 ( 15 ) ， 分割為 數個 溝槽 空間 ( 400 ) ， 能 降低 皂液 ( 3 ) 的 混合率 ， 且 藉 由此 種 設置 ， 更能 方便 使用者 組裝 與 拆卸 。 
 其次 ， 透過 控制 凹槽 ( 23 ) 之間 的 間隔 ， 還能 產生 多種 的 層次 變化 ， 讓 皂液 ( 3 ) 的 變化 能 更 多樣化 ， 以 產生 更 多種類 的 渲染 皂 ( 30 0 ) 。 
 上述 中 ， 所述 隔板 骨架 ( 21 ) 及 隔板 ( 22 ) 的 材質 ， 是 為 下列 之一 ： 壓克力 材 、 硬質 塑膠 材 。 
 其中 ， 以 不同 材質 來 做 為 隔板 骨架 ( 21 ) 及 隔板 ( 22 ) 之用 ， 能 增加 適用 的 範圍 ， 讓 使用者 能 有 更 多的 選擇 。 
 上述 中 ， 所述 皂模 ( 1 ) 的 開口 ( 11 ) ， 更 設有 一能將 第二 容 置空間 ( 15 ) 封閉的 頂蓋 ( 16 ) 。 
 其中 ， 藉由 頂蓋 ( 16 ) 的 設置 ， 能將 開口 ( 11 ) 封閉 ， 以 使 第二 容 置空間 ( 15 ) 與 外界 隔離 ， 能 使 其 內的 皂液 ( 3 ) ， 能夠 保溫 ， 並 降低 環境因素 的 影響 ， 讓 皂液 ( 3 ) 完美 的 變成 初始 渲染 皂 ( 200 ) 。

 第二 實施 型態 ： 
 圖式 中 揭示出 ， 為 一種 渲染 皂 成形工具 ( 100 ) ， 其特 徵在於 包括 ： 一 頂端 為 開口 ( 11 ) 的 皂模 ( 1 ) ， 該皂模 ( 1 ) 是 由 一 底座 ( 17 ) 、 及 一端 設 於 該 底座 ( 17 ) 內 、 呈 中空狀 、 可拆卸 式 的 一立壁 殼體 ( 18 ) 所 組成 ， 該 底座 ( 17 ) 與 立壁 殼體 ( 18 ) ， 圍繞 形成 一 第三 容納 空間 ( 19 ) ， 而 該 立壁 殼體 ( 18 ) 的 兩頂 相 對端 ， 各 設有 一 呈 齒狀的 卡接部 ( 18 1 ) ； 以及 一設 於 該 第三 容 置空間 ( 19 ) 內的 分隔 裝置 ( 2 ) ， 該 分隔 裝置 ( 2 ) 是 由 至少 一個以上 設 於 該 第三 容 置空間 ( 19 ) 內 、 兩端 分別 連接 卡接部 ( 18 1 ) 、 並能將 第三 容 置空間 ( 19 ) 分為 數個 溝槽 空間 ( 400 ) 的 隔板 ( 22 ) 所 組成 。 
 其中 ， 使用 另 一種 皂模 ( 1 ) 設置 ， 並 配合 分隔 裝置 ( 2 ) 的 使用 ， 除了 能 方便 渲染 外 ， 更能 再 進一步的 降低 整體的 成本 與 使用 難度 ， 有效的 提供 消費者 更 多的 選擇 。 
 其次 ， 透過 控制 卡接部 ( 18 1 ) 的齒 狀構造 ， 還能 產生 多種 的 層次 變化 ， 讓 皂液 ( 3 ) 的 變化 能 更 多樣化 ， 以 產生 更 多種類 的 渲染 皂 ( 30 0 ) 。 
 上述 中 ， 所述 所述 皂模 ( 1 ) 的 材質 ， 是 為 下列 之一 ： 壓克力 材 、 硬質 塑膠 材 ； 而 隔板 ( 22 ) 的 材質 ， 是 為 下列 之一 ： 壓克力 材 、 硬質 塑膠 材 。 
 其中 ， 以 不同 材質 來 做 為 皂模 ( 1 ) 及 隔板 ( 22 ) 之用 ， 能 增加 適用 的 範圍 ， 提供 更 多的 選擇 ， 讓 使用者 能 更 高的 購買意願 。 
 上述 中 ， 所述 該 隔板 ( 22 ) 與 卡接部 ( 18 1 ) 的 連接處 ， 設有 能 與 卡接部 ( 18 1 ) 配合 、 並 凸出 於 皂模 ( 1 ) 的 凸 楯 ( 24 ) 。 
 其中 ， 透過 凸 楯 ( 24 ) 的 設置 ， 讓 隔板 ( 22 ) 能 不 被 倒入 的 皂液 ( 3 ) 影響 ， 能 穩固的 將 第二 容 置空間 ( 15 ) ， 分割為 數個 溝槽 空間 ( 400 ) ， 能 降低 皂液 ( 3 ) 的 混合率 ， 且 藉 由此 種 設置 ， 更能 方便 使用者 組裝 與 拆卸 ， 而 不致 影響到 後續 的 作業 。 
 上述 中 ， 所述 皂模 ( 1 ) 的 開口 ( 11 ) 處 ， 更 設有 一能將 第三 容 置空間 ( 19 ) 封閉的 頂蓋 ( 16 ) 。 
 其中 ， 藉由 頂蓋 ( 16 ) 的 設置 ， 能將 開口 ( 11 ) 封閉 ， 以 使 第三 容 置空間 ( 19 ) 與 外界 隔離 ， 能 使 其 內的 皂液 ( 3 ) ， 能夠 保溫 ， 並 降低 環境因素 的 影響 ， 讓 皂液 ( 3 ) 完美 的 變成 初始 渲染 皂 ( 200 ) 。
